SVG

Scalable Vector Graphics

General info

SVG is declarative markup language for describe scalable vector graphics.

  • Open standard.
  • Developed by the W3C since 1999.
  • XML-based.
  • It is format for 2D-graphics. 3D is not supported.
  • Current version 1.1, released in 2011.

Path

Most flexible SVG element for curves drawing.

The element <path> is the most flexible SVG element for drawing open and closed curves. The <path> element has the attribute «d», inside of which there is list of commands. The commands define the trajectory and direction of the curly line.

Path

Moveto, lineto, horizontal lineto, vertical lineto.
Uppercase — absolute, lowercase — relative. Z — close stroke.
